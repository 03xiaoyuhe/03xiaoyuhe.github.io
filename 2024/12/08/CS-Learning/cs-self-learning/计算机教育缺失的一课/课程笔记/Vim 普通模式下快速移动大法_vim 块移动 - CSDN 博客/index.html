VimTutor

lessen 1

使用 h (left) j (down) k (up) l(right)进行移动
vim FILENAME <enter> 可以使用 vim 打开文件在标准模式下 输入 :q! 可以退出文件，不保存任何修改在标准模式下输入 :wq 会保存文件并退出在标准模式下按下 x 可以删除选中的字母按下 i 可以将模式切换为插入模式输入 A 在行尾输入内容

Lesson 2

dw 从当前位置删除到下一个单词开头
de 从当前位置删除到本单词末尾
d$ 从当前位置删除到行尾
dd 删除整行
[num]w 重复移动操作 num 遍修改命令的格式为 operator [number] motion:

  • opertor - 表示要做什么 例如 d 用来删除

  • [number] - 表示下面的移动操作的执行数量

  • motion - 表示移动的动作，例如 w 词，e 单词的结尾，$ 行的结尾，等等。将光标移动到行首 0
    撤销操作:

  • u 撤销一步

  • U 撤销本行所有修改

  • 重做 ctrl-r

Lesson 3

  1. p 在光标之后放置之后删除或复制的文本

  2. 键入 r 然后输入你想替换的字母，可以替换掉光标下的字母

  3. 当想对某部分内容进行更改时，可以键入 c 以及移动操作符，以更改想要更改的地方，如 ce 会更改从光标到词尾的内容，c$ 会更改从光标到行尾的内容

  4. 更改命令的格式为: c [number] motion

Lesson 4

  1. ctrl-g 会显示你当前在文件中的位置(行数)
    G 会将光标移动到文件的底部
    number G 会将光标移动到指定的行
    gg 会将光标移动到文件的首行

  2. 键入 / 后输入想要查找的文本，会从头查找所输入内容键入 ? 后输入想要查找的文本，会从文章的末尾开始查找输入内容

  3. 当光标在 ( , ), [, ], {, } 上时键入 % 后光标会移动到对应的括号

  4. :s/old/new 可以将第一个 old 字符替换为 new 字符
    :s/old/new/g 可以将文件中所有 old 字符替换为 new 字符
    :#,#s/old/new/g # 可以用来指明范围
    :%s/old/new/g 会替换掉所有出现在文件中的字符
    :%s/old/new/gc 加上 c 后,则会在每一次替换时都会询问

Lesson 5

  1. :!command 可以运行一个外部命令一些有用的命令：
    (Windows) (Unix)
    :!dir :!ls 展示文件列表
    :!del :!rm 删除文件

  2. w FILENAME 将整个 vim 文件另存为 FILENAME 文件

  3. v 进入选择模式，通过移动扩大选区，键入 :w FILENAME ：将选区另存为 FILENAME

  4. :r FILENAME 可以将 FILENAME 中的内容插入到光标所在位置之前。

  5. :r !dir 可以将 dir 命令的输出添加到光标之前

Lesson 6

  1. 键入 o ，在光标所在行的下一行，打开(open)一个新行键入 O ，在光标所在行的上一，行打开一个新行

  2. 键入 a 在光标之后插入内容键入 A 在行的末尾插入内容

  3. e (command)会将光标移动到单词的末尾

  4. y (operator) 会复制(yanks)选区的内容，p 会放置(puts)内容

  5. 键入 R 进入替换模式

  6. 键入 :set xxx 打开 xxx 选项。常见的选项有：
    ic / ignorecase 在查找时，忽略大小写的情况
    is / incsearch 展示所搜索字符的部分匹配
    hls / hlsearch 高亮显示所有的匹配

  7. 在所有选项前加 no 可以关闭选项，如：set noic

Lesson 7

  1. 键入 :help 打开帮助页面

  2. 键入 :help cmd 以获取对应命令的帮助

  3. ctrl-w 跳到另一个窗口

  4. 键入 :q 退出帮助文档

  5. 创建一个 vimrc 文件来保存初始运行设置

  6. 键入一个 : 命令，按下 ctrl-d 查看可能的匹配，按下 <TAB> 使用一个匹配。

for vimrc

vimrc 中存放了所有 : 命令，以设置编辑器的初始状态

一些常用的跳转操作

% 跳转到相配对的括号
gD 跳转到局部变量的定义处
'' 跳转到光标上次停靠的地方, 是两个’, 而不是一个"
mx 设置书签,x 只能是 a-z 的26个字母
`x 跳转到书签处("`"是1左边的键)
>增加缩进,"x>"表示增加以下 x 行的缩进
< 减少缩进,"x<"表示减少以下 x 行的缩进
{ 跳到上一段的开头
} 跳到下一段的的开头
( 移到这个句子的开头
) 移到下一个句子的开头
[[ 跳转至上一个函数(要求代码块中’{‘必须单独占一行)
]] 跳转至下一个函数(要求代码块中’{'必须单独占一行)
C-] 跳转至函数或变量定义处
C-O 返回跳转前位置
C-T 同上
nC-T 返回跳转 n 次
0 数字0,跳转至行首
^ 跳转至行第一个非空字符
$ 跳转至行尾
