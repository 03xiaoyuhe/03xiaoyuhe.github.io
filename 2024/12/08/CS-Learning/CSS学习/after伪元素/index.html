::after (:after)

CSS伪元素::after用来创建一个伪元素，作为已选中元素的最后一个子元素。通常会配合content属性来为该元素添加装饰内容。这个虚拟元素默认是行内元素。

[!info] 备注： CSS 引入 ::after 表示法（两个冒号）是用来区分伪类和伪元素的。同时为了向后兼容，浏览器也支持前文介绍的 :after。
