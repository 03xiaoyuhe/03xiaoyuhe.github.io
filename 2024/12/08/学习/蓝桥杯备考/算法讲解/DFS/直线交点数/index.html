题目描述

假设平面上有 $N$ 条直线，且无三线共点，那么这些直线有多少种可能的交点数？

输入格式

一行，一个整数 $N$，代表有 $N$ 条直线。

输出格式

一行，一个整数，表示方案总数。

样例输入 #1

样例输出 #1

提示

对于所有数据，满足 $1 \le N \le 25$。

题解

由于不存在三线共点，故只会存在相互平行，或者相互必产生一交点两种情况。

故，可以通过讨论共有几条线平行的情况来解决

通过分析

  1. 当边数为 0 时

    1. 此时交点数为 0，只有一种情况

  2. 当边数为 1 时

    1. 此时只有一条直线相互平行一种情况，剩下 0 条边，转化为 0 条边的情况，每种情况的交点数为：平行边 1 条和其余边的交点数，加上 0 条边情况构造的交点数

  3. 当边数为 2 时

    1. 当只有一条直线相互平行，即自己和自己平行时，剩下 1 条边，转化为 1 条边的子情况，每种子情况的交点数为：平行边 1 条和其余边的交点数，加上 1 条边当前子情况的交点数，
    2. 当有两条直线相互平行时，剩下 0 条边，转化为 0 条边的情况，

以此类推，同时为防止交点数统计重复，用 flag 数组标记已经统计过的点；
