在使用 lwIP 向特定 IP 地址发送 HTTP 请求时，需要通过 lwIP 提供的 TCP/IP 接口创建一个客户端套接字并实现 HTTP 协议的基本逻辑。以下是具体步骤和示例代码。

实现步骤

  1. 初始化 lwIP

    • 确保 lwIP 已初始化，包括网络接口配置和任务调度。

  2. 创建 TCP 连接

    • 使用 socket API 创建客户端套接字。
    • 连接到指定的 IP 和端口（通常 HTTP 使用 80 端口）。

  3. 发送 HTTP 请求

    • 根据 HTTP 协议格式构造请求头。
    • 将请求通过 send() 发送给目标服务器。

  4. 接收响应

    • 使用 recv() 接收服务器返回的数据并处理。

  5. 关闭连接

    • 释放套接字资源。

注意事项

  1. 网络配置

    • 确保设备正确配置了 IP 地址、子网掩码和网关（静态或 DHCP）。
    • 网络连通性可通过 ping 测试。

  2. DNS 支持

    • 如果使用域名而不是 IP 地址，可以结合 lwIP 的 DNS 功能解析域名。

  3. 错误处理

    • 检查 socket、connect、send、recv 的返回值，并进行适当的错误处理。

  4. 内存限制

    • 嵌入式设备内存有限，接收缓冲区不宜过大。如果响应内容较大，可以分段接收。

  5. HTTP 请求扩展

    • 修改 request 字符串可支持 POST、PUT 等方法，以及设置自定义的 HTTP 头字段。

调试和验证

  1. 抓包分析

    • 使用 Wireshark 抓包分析请求是否正确发送，响应是否符合预期。

  2. HTTP 服务端

    • 在目标服务器上运行 HTTP 服务以测试请求处理（例如使用 Python 简单 HTTP 服务器）。

通过以上代码和配置，您可以使用 lwIP 向指定 IP 地址发送 HTTP 请求并接收响应。

内置插件

出乎很多人的意料，Vim自己默认加载了大把大把的插件。启动 Vim 后，用:scriptnames查看所有加载的源文件

这些文件多数永远不会被用到，所以请禁止如你所见的以下插件。这些插件仍旧会在源文件列出，但是当Vim启动时，只有第一行会被读取，而之后的配置，命令和日志都不会执行。

插件 禁用 帮助
2html let g:loaded_2html_plugin = 1 :h 2html
getscript let g:loaded_getscriptPlugin = 1 :h pi_getscript
gzip let g:loaded_gzip = 1 :h pi_gzip
logipat let g:loaded_logipat = 1 :h pi_logipat
matchparen let g:loaded_matchparen = 1 :h pi_paren
netrw let g:loaded_netrwPlugin = 1 :h pi_netrw
rrhelper let g:loaded_rrhelper = 1 :e $VIMRUNTIME/plugin/rrhelper.vim
spellfile let g:loaded_spellfile_plugin = 1 :h spellfile.vim
tar let g:loaded_tarPlugin = 1 :h pi_tar
vimball let g:loaded_vimballPlugin = 1 :h pi_vimball
zip let g:loaded_zipPlugin = 1 :h pi_zip

映射大写锁定键（CapsLock）到控制键（Ctrl）

大写锁定键属于键盘上最少用到的按键，但是它要比控制键更加容易按到，因为它位于你手放在键盘时的起始行。如果你敲很多代码的话，映射大写锁定键（CapsLock）到控制键（Ctrl）可以有效降低重复使力伤害 (ps:为什么将 jk 映射为 esc，而不是 jj，重复点击同一个按键对手指伤害很大)。

友情提示：当你习惯这个设定后，你就离不开它了。

OSX:

System Preferences -> Keyboard -> Keyboard Tab -> Modifier Keys. 变更
“CapsLock” 为 “Control”。

Linux:

为了使按键生效，请把以下几行放入 ~/.xmodmap 文件:

remove Lock = Caps_Lock
keysym Caps_Lock = Control_L
add Control = Control_L

用以下命令使之生效
$ xmodmap ~/.xmodmap。

替代方法有caps2esc 或者
xcape。

Windows:

请参看 superuser.com: 在Windows8.1映射大写锁定键到控制键。
